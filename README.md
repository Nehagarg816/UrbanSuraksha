  <details>

    <ul>
      <li>ReactJS: Powering dynamic and interactive UI.</li>
      <li>Tailwind CSS: Utility-first CSS for custom styling.</li>
      <li>JavaScript: Logic and event handling.</li>
    </ul>

  <details>
    <summary><strong>Backend</strong></summary>
    <ul>
      <li>NodeJS: Backend runtime for server operations.</li>
      <li>ExpressJS: Routing and middleware support.</li>
      <li>MongoDB: NoSQL database for scalable storage.</li>
    </ul>
  </details>

  <details>
    <summary><strong>Cloud & Tools</strong></summary>
    <ul>
      <li>MongoDB Atlas: Cloud-hosted database for storing PG, user, reviews, and bill data.</li>
      <li>Cloudinary: For image uploads in PG listings.</li>
      <li>Google Maps API: Integrated map view of PGs for easy location-based searches.</li>
      <li>LeafletJS: Interactive map for locating tiffin services.</li>
      <li>Web Speech API: Voice-to-text conversion for complaint registration.</li>
    </ul>
  </details>
